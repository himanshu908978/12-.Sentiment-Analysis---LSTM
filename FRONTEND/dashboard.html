Project 12 :- Sentiment Analysis

About this project

This is my Tweleth project in the domain of AI/ML/DL whose AI model is made by using the Pytorch and concepts of NLP(Natural Language Processing) and trained on a tweet dataset dataset.

This Sentiment analysis project detects whether there is an positiveness or negativeness is in your given sentiment.

This trained AI model detect the labels in sentiment and gives you a label that it could find in your given sentiment and also give a Confidence score that represents that how much confident model really is to give you that output.

Labels :-

  • Negative
  • Positive

So if you upload a sentiment then you will get a predicted class from model in this scenario positive and negative label and confidence score that shows the surety of predicted class.

Conclusion

I trained this model on google colab notebook's free GPU T4(it comes with a specfic duration of usage) for 20 epochs and then the minimum loss was about 0.19 something and i got 92% accuracy on testing dataset of tweeter tweet dataset.

Test dataset metrics :-

  • 1).Accuracy :- 0.8108
  • 2).F1 :- 0.8108

So final conclusion is that this model can predict the positiveness or negativeness of sentiment .
